Description [src]

Computes the logical width of the glyph string.

This can also be computed using pango_glyph_string_extents(). However, since this only computes the width, it’s much faster. This is in fact only a convenience function that computes the sum of geometry.width for each glyph in the glyphs.

Return value

Returns: int

the logical width of the glyph string.
